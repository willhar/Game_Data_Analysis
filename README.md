# Steam Games & Player Count Analysis

## Overview
This project explores trends in the Steam gaming ecosystem by analyzing player activity and game metadata. The goal of the analysis is to identify patterns in player engagement, genre popularity, and long-term trends across the Steam platform.

## Datasets Used
This project uses two datasets from Kaggle:

- **Steam Monthly Average Players Dataset**  
  Source: https://www.kaggle.com/datasets/lunthu/steam-monthly-average-players  
  Contains historical monthly average player counts for Steam games.

- **Steam Games Dataset**  
  Source: https://www.kaggle.com/datasets/fronkongames/steam-games-dataset  
  Contains metadata such as game titles, genres, release dates, and other attributes.
